## About
Arabic Firefox extension for controlling video/audio playback speed across all websites.

### Features
- Smooth speed control (0.25x - 4x)
- Keyboard shortcuts (Alt + Up/Down)
- Auto-save preferred speed
- Works on all websites
- Privacy-focused (no data collection)
